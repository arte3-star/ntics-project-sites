# Escritório de Projetos (PMO)

- **Responsável:** [preencher]
- **Missão:** Estruturar, abrir, executar e comunicar projetos NTICS com excelência — desde o Termo de Abertura até a entrega e divulgação, garantindo rastreabilidade e impacto documentado.
- **Workflows:** [workflows/escritorio-projetos/](../../workflows/escritorio-projetos/)
- **Tools:** [tools/integrations/](../../tools/integrations/), [tools/meetings/](../../tools/meetings/), [tools/adobe/](../../tools/adobe/), [tools/publishing/](../../tools/publishing/)
- **Squads:** N/A

## Fluxo de Projetos

```
Perfil Estratégico do Patrocinador
  └→ Termo de Abertura (TA)
       ├→ Plano de Divulgação
       │    └→ /briefing-video (roteiro + carrossel)
       │         └→ Briefing Website → /criar-site
       └→ Processamento de Reuniões → tasks ClickUp
```
