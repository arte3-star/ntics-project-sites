# Feedback e Aprendizados

> Cada regra tem: o que fazer, por que (o que aconteceu), e quando aplicar.

---

## Comunicação

**Regra:** Sempre gere rascunho de e-mail para aprovação, nunca envie direto.
**Por que:** Na primeira semana, o assessor sugeriu enviar um e-mail com tom errado para um cliente importante.
**Quando aplicar:** Toda vez que houver qualquer ação de comunicação externa.

---

## ClickUp / Tasks

**Regra:** Após criar task no ClickUp, confirme que existe com get_task antes de dizer que está pronto.
**Por que:** O 200 OK da criação não garante que a task foi criada corretamente.
**Quando aplicar:** Toda operação de criação em ClickUp, Gmail, Drive.
