# Workflow: Roteiro de Vídeo por Fase de Projeto

## Objetivo
Gerar roteiros prontos para editor de vídeo para qualquer fase do projeto (pré, durante, pós/case, venda), a partir de release, TAP ou briefing.

## Quando Executar
Quando precisar de vídeo de divulgação, acompanhamento, case ou venda para um projeto NTICS.

## Inputs Necessários

| Input | Fonte | Obrigatório |
|-------|-------|-------------|
| Release, TAP ou briefing do projeto | Área de comunicação ou workflows anteriores | Sim (pelo menos 1) |
| Tipo do vídeo | Coordenador escolhe: Pré / Durante / Pós (Case) | Sim |
| KV do Projeto | Link Google Drive | Sim |
| Logo do Patrocinador | Link Google Drive | Sim |
| Vídeos/fotos do projeto | Pasta Google Drive | Recomendado |
| Referência de design | Instagram do patrocinador (URL) | Recomendado |
| Depoimento captado (durante/pós) | Coordenador informa se existe | Condicional |

## Regras Críticas

- O release manda nos fatos (nomes, cidades, números, datas, patrocinador, públicos)
- Nunca inventar fatos. Dado ausente → [PREENCHER]
- **Pré, Durante e Case — Nunca citar:** Ministério da Cultura, Lei Rouanet, Lei de Incentivo, incentivo fiscal
- **Reels Case (Impacto e Valor) — PODEM citar:** Lei de Incentivo à Cultura, RSC, ODS, incentivo fiscal (são vídeos institucionais de captação)
- **Cartela final:** Logo [PATROCINADOR] + NTICS Projetos
- **Quem comunica:** Sempre a NTICS, citando o patrocinador
- **Fórmula do patrocinador:** "Com patrocínio de [empresa]" (Pré, Durante, Case)
- **Reels Case:** NÃO citar nome do patrocinador nem cidades. Números da edição contextualizados ("em uma única edição", "só nessa ação"). Depoimento obrigatório.
- **Contagem de palavras:** Obrigatória. Narração PT-BR confortável = ~2,3 palavras/segundo
- Após gerar cada roteiro, contar palavras por fase. Se ultrapassar o budget, reescrever mais curto.

## 5 Tipos de Vídeo

| Tipo | Duração | Formato | Destinatário | CTA |
|------|---------|---------|--------------|-----|
| Pré-projeto | 60s | Vertical 9:16 | Redes sociais (collab patrocinador) | "Siga nas redes" |
| Durante | 60s | Vertical 9:16 | Redes sociais (collab patrocinador) | "Acompanhe" |
| Pós-projeto (Case) | 90–120s | **Horizontal 16:9** | **Entrega para o patrocinador** | Institucional (sem captação) |
| Reels Case Impacto | 60s | Vertical 9:16 | Captação de patrocinadores | "Fale com a NTICS" |
| Reels Case Valor | 60s | Vertical 9:16 | Captação de patrocinadores | "Fale com a NTICS" |

**Nota:** Quando o coordenador pede "pós-projeto", gerar automaticamente os 3 roteiros: Case + Venda 1 + Venda 2.

## Arquitetura de Tom

| Dimensão | Pré | Durante | Case (Pós) | Reels Case (Impacto/Valor) |
|----------|-----|---------|------------|----------------------------|
| Quem fala | NTICS (institucional) | NTICS comunica | NTICS (storytelling) | NTICS (institucional captação) |
| Tempo verbal | Futuro: "vai acontecer" | Presente: "está acontecendo" | Passado: "transformou" | Passado + convite |
| Menção patrocinador | "Com patrocínio de X" | "Com patrocínio de X" | "Com patrocínio de X" | NÃO cita patrocinador |
| Cita cidade | Sim | Sim | Sim | NÃO cita cidades |
| Lei de Incentivo/RSC | Proibido | Proibido | Proibido | **Pode e deve citar** |
| Emoção | Antecipação, confiança | Energia, autenticidade | Orgulho, emoção | Inspiração + oportunidade |
| Dados | Escala prometida | Atividades em andamento | Resultados alcançados | "Em uma única edição" / "Só nessa ação" |
| Depoimento | Não | Se houver | Se houver | **Obrigatório** |
| Fechamento | Logo NTICS + Patrocinador | Logo NTICS + Patrocinador | Logo Patrocinador + NTICS | Logo NTICS (sem patrocinador) |

## Passo a Passo

### Passo 1 — Identificação
Extrair do material:

| Campo | Exemplo |
|-------|---------|
| Nome do Projeto | Cultura Robótica |
| Nº do Projeto | 116 |
| Cidade/UF | Campo Grande/MS + Sidrolândia/MS |
| Patrocinador | Áster |
| Realização | NTICS Projetos |
| Coordenação/Produção | [nome se constar] |
| Público-alvo | Estudantes e professores da rede pública |
| Tema/Solução | [descrição curta] |
| CSR / Impacto Social | [palavras-chave] |
| KV do Projeto | [link] |
| Logo Patrocinador | [link] |
| Vídeos | [link pasta] |
| Referência de Design | [link Instagram patrocinador] |

### Passo 2 — Definir tipo do vídeo
Perguntar ao coordenador: **Pré-projeto, Durante ou Pós-projeto (Case)?**

Se Durante ou Pós: **"Já tem depoimento captado de participante?"**
- Se sim: "Qual é a fala (resumida)?" → reservar ~10s no roteiro para a fala (não narrada, só legenda)
- Se não: fase de participantes usa narração sobre eles

### Passo 3 — Gerar o roteiro
Usar o template correspondente ao tipo (seções abaixo).

### Passo 4 — Contar palavras
Após gerar, contar palavras de cada fase. Se alguma fase ultrapassar o budget, reescrever.

### Passo 5 — Entregar
Saída: **Ficha do Projeto** (tabela) + **Resumo** (1 parágrafo) + **Formato** + **Tabela do Roteiro** com coluna de contagem de palavras.

## Templates por Tipo

### TIPO 1: Pré-Projeto (60s, max 140 palavras, Vertical 9:16)

| Fase | Tempo | Seg | Budget | Script | Imagem (B-roll) | Gráficos/Áudio |
|------|-------|-----|--------|--------|-----------------|-----------------|
| 1. Abertura | 0:00–0:10 | 10s | ~22 pal | Apresentar projeto + "com patrocínio de [empresa]" + cidades | Planos das cidades, escolas, espaços | Nome do projeto + cidades |
| 2. Propósito | 0:10–0:22 | 12s | ~28 pal | Por que o projeto existe, o que propõe | Público em atividade, materiais | Palavras-chave em cards |
| 3. Como vai acontecer | 0:22–0:40 | 18s | ~42 pal | Frentes, atividades, números de impacto previsto | Oficinas, dinâmicas, atividades | Cards com números |
| 4. Convite/Chamada | 0:40–0:50 | 10s | ~23 pal | CTA: acompanhar, participar, inscrever | Pessoas chegando, interação | Data/local se houver |
| 5. Fechamento | 0:50–1:00 | 10s | ~15 pal | Frase de visão + redes sociais | Mosaico de momentos | Cartela: Patrocinador + NTICS |

### TIPO 2: Durante (60s, max 140 palavras, Vertical 9:16)

| Fase | Tempo | Seg | Budget | Script | Imagem (B-roll) | Gráficos/Áudio |
|------|-------|-----|--------|--------|-----------------|-----------------|
| 1. Abertura | 0:00–0:10 | 10s | ~22 pal | "Está acontecendo o [projeto], com patrocínio de [empresa], em [cidade]" | Atividades acontecendo, escola, espaço | Nome do projeto + cidade |
| 2. Atividades | 0:10–0:25 | 15s | ~35 pal | O que está sendo realizado: oficinas, encontros, atividades do dia | Imagens das atividades, bastidores, interação | Palavras-chave |
| 3. Participantes | 0:25–0:42 | 17s | ~40 pal | Quem está participando, reações, experiência. Se tem depoimento: ~10s para fala + legenda | Rostos, interação, depoimento em vídeo | Legenda do depoimento se houver |
| 4. Próximas atividades + CTA | 0:42–0:52 | 10s | ~23 pal | O que vem pela frente + "acompanhe nas redes" | Preparação, próximos passos | Redes sociais |
| 5. Fechamento | 0:52–1:00 | 8s | ~10 pal | Frase curta de encerramento | Imagem de energia | Cartela: NTICS + Patrocinador |

---

### TIPO 3: Pós-Projeto / Case (90–120s, max 280 palavras, Horizontal 16:9) — ENTREGA PARA PATROCINADOR

**IMPORTANTE:** Este vídeo é entrega para o patrocinador. NÃO usa CTA de captação ou "conheça mais projetos". Fechamento institucional com logos.

| Fase | Tempo | Seg | Budget | Script | Imagem (B-roll) | Gráficos/Áudio |
|------|-------|-----|--------|--------|-----------------|-----------------|
| 1. Gancho emocional | 0:00–0:15 | 15s | ~35 pal | Frase de impacto que prende atenção. Cena forte. | Momento marcante do projeto | Trilha emocional |
| 2. Contexto | 0:15–0:35 | 20s | ~47 pal | O que foi o projeto, onde aconteceu, "com patrocínio de [empresa]" | Imagens das cidades, espaços, início das atividades | Nome do projeto + cidades + patrocinador |
| 3. Jornada | 0:35–0:55 | 20s | ~47 pal | Como aconteceu: etapas, atividades, metodologia | Sequência cronológica das atividades | Linha do tempo visual |
| 4. Impacto | 0:55–1:15 | 20s | ~47 pal | Números concretos: pessoas impactadas, cidades, escolas, indicadores | Imagens de escala, grupos grandes | Cards com números |
| 5. Transformação | 1:15–1:35 | 20s | ~47 pal | Depoimento de participante (~15s fala + ~5s narração). Se não tem depoimento forte, usar 2-3 falas curtas. | Rosto do depoente, cenas de transformação | Legenda da fala |
| 6. Fechamento | 1:35–2:00 | 25s | ~55 pal | Legado do projeto, visão de futuro, encerramento institucional | Mosaico final, momentos de celebração | Cartela: Patrocinador + NTICS |

---

### TIPO 4: Reels Case Impacto (60s, max 140 palavras, Vertical 9:16) — CAPTAÇÃO DE PATROCINADORES

Tom institucional NTICS. Fala do projeto como um todo. NÃO cita patrocinador nem cidade. PODE citar Lei de Incentivo, RSC, ODS. Números da edição contextualizados. Depoimento obrigatório.

| Fase | Tempo | Seg | Budget | Script | Imagem (B-roll) | Gráficos/Áudio |
|------|-------|-----|--------|--------|-----------------|-----------------|
| 1. Gancho | 0:00–0:08 | 8s | ~16 pal | Número forte da edição: "Em uma única edição, [X] estudantes..." | Cena de maior impacto | Número em destaque |
| 2. O projeto | 0:08–0:20 | 12s | ~26 pal | O que o projeto faz (genérico, sem cidade nem patrocinador) | Atividades principais | Nome do projeto |
| 3. Depoimento | 0:20–0:32 | 12s | ~20 pal | [DEPOIMENTO — legendado] Fala real de participante | Rosto do participante | Legenda da fala |
| 4. Impacto + RSC | 0:32–0:48 | 16s | ~36 pal | "Só nessa ação, [X] alunos e [Y] professores. Viabilizado pela Lei de Incentivo à Cultura, que transforma RSC em impacto real." | Formação, mosaico, indicadores | Cards números + ODS |
| 5. CTA captação | 0:48–1:00 | 12s | ~20 pal | "Sua empresa pode protagonizar a próxima edição. Fale com a NTICS." | Logo NTICS + contato | Logo NTICS |

---

### TIPO 5: Reels Case Valor (60s, max 140 palavras, Vertical 9:16) — CAPTAÇÃO DE PATROCINADORES

Mesmo material do Tipo 4, ângulo diferente. Foco: mostrar o valor para a empresa que decidir patrocinar. Mesmas regras (sem patrocinador, sem cidade, pode citar Lei de Incentivo e RSC).

| Fase | Tempo | Seg | Budget | Script | Imagem (B-roll) | Gráficos/Áudio |
|------|-------|-----|--------|--------|-----------------|-----------------|
| 1. Gancho | 0:00–0:08 | 8s | ~18 pal | Pergunta provocativa: "E se a sua empresa pudesse investir em [tema] usando a Lei de Incentivo à Cultura?" | Cena impactante | Texto da pergunta |
| 2. O que é o projeto | 0:08–0:22 | 14s | ~25 pal | O que o projeto faz e quem forma (genérico, institucional) | Atividades principais | Palavras-chave |
| 3. Depoimento | 0:22–0:34 | 12s | ~26 pal | [DEPOIMENTO — legendado] + "Em cada edição, histórias como essa se multiplicam." | Rosto, transição para mosaico | Legenda da fala |
| 4. Valor + RSC | 0:34–0:48 | 14s | ~29 pal | "Só na última edição, [X] impactados. Patrocinar é associar a sua marca a RSC e aos ODS da ONU." | Mosaico de impacto | Cards números + ODS |
| 5. CTA captação | 0:48–1:00 | 12s | ~17 pal | "Transforme incentivo fiscal em impacto social. Fale com a NTICS." | Logo NTICS + contato | Logo NTICS |

Para cada tipo, entregar:

```
========================================
FICHA DO PROJETO
========================================
Nome: [nome]
Nº: [número]
Patrocinador: [empresa]
Cidade/UF: [lista]
Público: [descrição]
Tema: [descrição curta]
KV do Projeto: [link]
Logo Patrocinador: [link]
Vídeos: [link pasta]
Referência de Design: [link Instagram]
========================================

TIPO DO VÍDEO: [Pré / Durante / Case / Venda 1 / Venda 2]
FORMATO: [Vertical 9:16 / Horizontal 16:9]
DURAÇÃO: [60s / 90-120s]

RESUMO (1 parágrafo)
[Parágrafo descritivo do projeto e do que o vídeo comunica]

========================================
ROTEIRO
========================================

| Fase | Tempo | Script | Palavras | Imagem (B-roll) | Gráficos/Áudio |
|------|-------|--------|----------|-----------------|-----------------|
| ... | ... | ... | ... | ... | ... |

========================================
CONTAGEM DE PALAVRAS
========================================
Fase 1: XX palavras (budget: ~YY)
Fase 2: XX palavras (budget: ~YY)
...
TOTAL: XX palavras (budget: ~YYY)
```

**Se tipo = Pós-projeto:** O Canvas inclui os 3 roteiros (Case + Venda 1 + Venda 2), cada um com sua própria tabela e contagem.

## Checklist de Qualidade

- [ ] Nome do projeto correto
- [ ] Patrocinador correto
- [ ] Cidades corretas
- [ ] Números corretos (do release/TAP)
- [ ] Sem Ministério da Cultura / Lei Rouanet / Lei de Incentivo
- [ ] Tom coerente com o tipo de vídeo
- [ ] Texto fluido para locução
- [ ] Contagem de palavras dentro do budget por fase
- [ ] Total de palavras dentro do limite (140 para 60s, 280 para 120s)
- [ ] Formato correto (vertical 9:16 ou horizontal 16:9)
- [ ] Fechamento com logos corretos para o tipo
- [ ] Depoimento incluído (se captado e tipo Durante/Pós)
- [ ] Reels Case sem citar nome do patrocinador nem cidades
- [ ] Reels Case com Lei de Incentivo, RSC e ODS citados
- [ ] Reels Case com depoimento incluído (obrigatório)
- [ ] CTA adequado ao tipo e destinatário
