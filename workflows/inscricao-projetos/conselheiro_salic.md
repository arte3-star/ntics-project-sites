# Workflow: Conselheiro SALIC MinC

## Objetivo
Editar e revisar textos campo-a-campo para colagem no sistema SALIC/MinC, entregando versões curta e completa, com revisão de coerência e conformidade.

## Quando Executar
Quando precisar preencher ou revisar campos do SALIC para inscrição/submissão de projetos de incentivo cultural.

## Inputs Necessários
| Input | Fonte | Obrigatório |
|-------|-------|-------------|
| Informações do projeto | Documentos do projeto, briefings, drafts | Sim |
| Manuais/referências MinC | Anexos na pasta de knowledge | Recomendado |
| Fase atual do projeto no SALIC | Usuário informa | Recomendado |

## Regras Críticas
- Base primária: anexos enviados + fontes oficiais (MinC/SALIC/IN/portarias)
- Citar de onde veio cada regra/orientação (anexo X ou link oficial)
- Se não houver base, não inventar: sinalizar lacuna e perguntar
- Saída padrão por campo: versão CURTA + versão COMPLETA, ambas em tom técnico/institucional
- Não incluir contagem de caracteres (apenas mencionar limite se documentado)
- Orçamento: só atuar quando solicitado explicitamente

## Passo a Passo

### Passo 1 — Confirmação de contexto
Confirmar com o usuário:
- Tipo de incentivo/sistema
- Fase atual (inscrição, execução, prestação de contas, diligência)
- Status do proponente
- Prazos
- UF/município
- Categoria/área cultural

### Passo 2 — Produção por campo
Para cada campo relevante do SALIC:
1. **Título do campo**
2. **Versão CURTA** — texto enxuto, direto
3. **Versão COMPLETA** — texto desenvolvido, institucional
4. **Notas de consistência** — pontos a conferir, documentos que sustentam, riscos de diligência

### Passo 3 — Checagens cruzadas
Verificar coerência entre:
- Objetivos ↔ Metas ↔ Etapas ↔ Cronograma ↔ Orçamento ↔ Plano de comunicação ↔ Acessibilidade ↔ Democratização/Contrapartidas

### Passo 4 — Preparação para diligências (quando solicitado)
1. Listar provável questionamento
2. Justificativa técnica
3. Texto de resposta por item

## Fases Típicas Cobertas
1. **Preparação:** elegibilidade, cadastro/credenciamento, documentação
2. **Concepção:** escopo, justificativa, objetivos, metas, público, território, impacto
3. **Planejamento:** etapas, cronograma, equipe/fichas técnicas
4. **Orçamento:** itens, memória de cálculo, coerência (quando solicitado)
5. **Políticas transversais:** acessibilidade, democratização, contrapartidas, sustentabilidade, comunicação
6. **Anexos:** documentos do proponente, autorizações, direitos autorais, cartas
7. **Revisão pré-envio:** consistência, linguagem, riscos, checklist
8. **Submissão e acompanhamento:** protocolo, status, diligências e respostas

## Output Esperado
Textos por campo do SALIC (versão curta + completa) + notas de consistência + respostas a diligências quando solicitado.

## Checklist de Qualidade
- [ ] Cada campo tem versão curta e completa
- [ ] Tom técnico/institucional mantido
- [ ] Fontes citadas (anexo ou link oficial)
- [ ] Checagem cruzada realizada (objetivos ↔ metas ↔ etapas)
- [ ] Riscos de diligência identificados
- [ ] Lacunas sinalizadas com perguntas objetivas
