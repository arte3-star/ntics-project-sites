# Templates de Roteiro de Vídeo — Knowledge Base

## Formato de saída (todos os tipos)

Sempre entregar neste formato:

```
FICHA DO PROJETO
========================================
| Campo | Conteúdo |
| Nome do Projeto | [nome] |
| Nº do Projeto | [número] |
| Cidade/UF | [lista] |
| Patrocinador | [empresa] |
| Realização | NTICS Projetos |
| Coordenação/Produção | [nome se constar] |
| Público-alvo | [descrição] |
| Tema/Solução | [descrição curta] |
| CSR / Impacto Social | [palavras-chave] |
| KV do Projeto | [link] |
| Logo Patrocinador | [link] |
| Vídeos | [link pasta] |
| Referência de Design | [link Instagram] |

RESUMO (1 parágrafo)
[Parágrafo descritivo]

FORMATO: [Vertical 9:16 / Horizontal 16:9]
DURAÇÃO: [60s / 90-120s]
```

Seguido da tabela do roteiro com colunas: Fase | Tempo | Script | Palavras | Imagem (B-roll) | Gráficos/Áudio

## TIPO 1: Pré-Projeto (60s, max 140 palavras, Vertical 9:16)

| Fase | Tempo | Seg | Budget |
|------|-------|-----|--------|
| 1. Abertura | 0:00–0:10 | 10s | ~22 palavras |
| 2. Propósito | 0:10–0:22 | 12s | ~28 palavras |
| 3. Como vai acontecer | 0:22–0:40 | 18s | ~42 palavras |
| 4. Convite/Chamada | 0:40–0:50 | 10s | ~23 palavras |
| 5. Fechamento | 0:50–1:00 | 10s | ~15 palavras |

### Orientações por fase

**Fase 1 — Abertura:** Apresentar o projeto e o patrocínio. Citar empresa + cidades. Ex: "Em [cidade], [projeto] chega com patrocínio de [empresa] para [público]."

**Fase 2 — Propósito:** Por que o projeto existe. O que propõe. Qual transformação busca.

**Fase 3 — Como vai acontecer:** Atividades, etapas, números de impacto previsto. Esta é a fase mais longa — usar com dados concretos.

**Fase 4 — Convite/Chamada:** CTA direto. "Acompanhe nas redes", "Inscrições abertas", "Saiba mais".

**Fase 5 — Fechamento:** Frase curta de visão + redes sociais. Cartela: Patrocinador + NTICS.

## TIPO 2: Durante (60s, max 140 palavras, Vertical 9:16)

| Fase | Tempo | Seg | Budget |
|------|-------|-----|--------|
| 1. Abertura (cita patrocinador) | 0:00–0:10 | 10s | ~22 palavras |
| 2. Atividades acontecendo | 0:10–0:25 | 15s | ~35 palavras |
| 3. Participantes + depoimento | 0:25–0:42 | 17s | ~40 palavras |
| 4. Próximas atividades + CTA | 0:42–0:52 | 10s | ~23 palavras |
| 5. Fechamento | 0:52–1:00 | 8s | ~10 palavras |

### Orientações por fase

**Fase 1 — Abertura:** "Está acontecendo o [projeto], com patrocínio de [empresa], em [cidade]."

**Fase 2 — Atividades:** O que está sendo realizado naquele momento. Oficinas, encontros, atividades.

**Fase 3 — Participantes:** Se tem depoimento captado → reservar ~10s para a fala (não narrada, só legenda) + ~7s de narração contextual. Se não tem depoimento → narração descritiva sobre os participantes.

**Fase 4 — Próximas atividades:** O que vem a seguir + "acompanhe nas redes @nticsprojetos e @[patrocinador]".

**Fase 5 — Fechamento:** Frase curta. Cartela: NTICS + Patrocinador.

## TIPO 3: Pós-Projeto / Case (90–120s, max 280 palavras, Horizontal 16:9)

**ENTREGA PARA O PATROCINADOR. Sem CTA de captação.**

| Fase | Tempo | Seg | Budget |
|------|-------|-----|--------|
| 1. Gancho emocional | 0:00–0:15 | 15s | ~35 palavras |
| 2. Contexto (patrocinador) | 0:15–0:35 | 20s | ~47 palavras |
| 3. Jornada | 0:35–0:55 | 20s | ~47 palavras |
| 4. Impacto (números) | 0:55–1:15 | 20s | ~47 palavras |
| 5. Transformação (depoimento) | 1:15–1:35 | 20s | ~47 palavras |
| 6. Fechamento | 1:35–2:00 | 25s | ~55 palavras |

### Orientações por fase

**Fase 1 — Gancho:** Frase de impacto que prende. Cena forte. Dado surpreendente ou momento marcante.

**Fase 2 — Contexto:** O que foi o projeto, onde aconteceu, "com patrocínio de [empresa]".

**Fase 3 — Jornada:** Cronologia das atividades, metodologia, como foi executado.

**Fase 4 — Impacto:** Números concretos: pessoas impactadas, cidades, escolas, indicadores.

**Fase 5 — Transformação:** Se tem depoimento → ~15s fala + ~5s narração. Se não → 2-3 falas curtas complementando números.

**Fase 6 — Fechamento:** Legado, visão de futuro, encerramento institucional. Cartela: Patrocinador + NTICS.

## TIPO 4: Reels Case Impacto (60s, max 140 palavras, Vertical 9:16) — CAPTAÇÃO DE PATROCINADORES

**Tom institucional NTICS. Fala do projeto como um todo, não de uma ação patrocinada.**

### Regras específicas dos Reels Case:
- **NÃO citar nome do patrocinador** — o projeto é apresentado como da NTICS
- **NÃO citar cidades específicas** — falar do projeto em geral
- **PODE citar:** Lei de Incentivo à Cultura, responsabilidade social corporativa, ODS, incentivo fiscal
- **Números de uma edição:** Contextualizar como "em uma única edição", "só nessa ação", "só na última edição"
- **Números acumulados NTICS** podem ser usados para reforço (11M+ impacto direto, 338K+ alunos, 244+ cidades)
- **Depoimento obrigatório** — dá veracidade ao conteúdo

| Fase | Tempo | Seg | Budget |
|------|-------|-----|--------|
| 1. Gancho (dado de impacto) | 0:00–0:08 | 8s | ~16 palavras |
| 2. O projeto (o que é) | 0:08–0:20 | 12s | ~26 palavras |
| 3. Depoimento (legendado) | 0:20–0:32 | 12s | ~20 palavras |
| 4. Impacto + RSC + ODS | 0:32–0:48 | 16s | ~36 palavras |
| 5. CTA captação | 0:48–1:00 | 12s | ~20 palavras |

### Orientações por fase

**Fase 1 — Gancho:** Número forte da edição + frase curta. Ex: "Em uma única edição, 8.100 estudantes descobriram que podem criar o futuro com as próprias mãos."

**Fase 2 — O projeto:** Descrição genérica do que o projeto faz (sem citar cidade nem patrocinador). Tom institucional.

**Fase 3 — Depoimento:** Fala real de participante, legendada. Se não houver, marcar [PREENCHER]. Essencial para dar veracidade.

**Fase 4 — Impacto + RSC:** Números da edição ("só nessa ação, X alunos e Y professores"). Citar Lei de Incentivo à Cultura, responsabilidade social corporativa e ODS relevantes.

**Fase 5 — CTA:** "Sua empresa pode protagonizar a próxima edição. Fale com a NTICS e transforme incentivo fiscal em impacto social."

## TIPO 5: Reels Case Valor (60s, max 140 palavras, Vertical 9:16) — CAPTAÇÃO DE PATROCINADORES

**Mesmo material do Tipo 4, ângulo diferente: foca no valor para a empresa que decidir patrocinar.**

### Mesmas regras do Tipo 4 (sem patrocinador, sem cidade, pode citar Lei de Incentivo e RSC).

| Fase | Tempo | Seg | Budget |
|------|-------|-----|--------|
| 1. Gancho (pergunta provocativa) | 0:00–0:08 | 8s | ~18 palavras |
| 2. O que é o projeto | 0:08–0:22 | 14s | ~25 palavras |
| 3. Depoimento (legendado) | 0:22–0:34 | 12s | ~26 palavras |
| 4. Valor para a empresa + RSC | 0:34–0:48 | 14s | ~29 palavras |
| 5. CTA captação | 0:48–1:00 | 12s | ~17 palavras |

### Orientações por fase

**Fase 1 — Gancho:** Pergunta provocativa. Ex: "E se a sua empresa pudesse investir em educação, inovação e sustentabilidade usando a Lei de Incentivo à Cultura?"

**Fase 2 — O projeto:** O que o projeto faz e quem forma. Tom institucional.

**Fase 3 — Depoimento:** Fala real + transição: "Em cada edição, histórias como essa se multiplicam."

**Fase 4 — Valor:** Números da edição ("só na última edição, X impactados"). Conectar a RSC, ODS, Agenda 2030. Ex: "Patrocinar é associar a sua marca a responsabilidade social corporativa e aos ODS da ONU."

**Fase 5 — CTA:** "Transforme incentivo fiscal em impacto social. Fale com a NTICS e traga esse projeto para a sua empresa."
