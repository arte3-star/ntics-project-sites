# Anexo 4 - Indicadores Camada

## KPIs Gerais NTICS (Master)
Padrões: **Local = município**, **Diretos = participante único por atividade**, **Indiretos = Diretos × 3**, **Alcance = Diretos + Indiretos**, **faixa etária completa**, **acessibilidade condicional (N/A quando não aplicável)**.
Escala padrão: **0–10**.

## Redação padrão das perguntas (para formulários)
* **Gênero (KPI_006):** "Qual seu gênero?" (Feminino / Masculino / Prefere não responder)
* **Faixa etária (KPI_005):** "Qual sua faixa etária?" (5–9 / 10–14 / 15–17 / 18–24 / 25–34 / 35–44 / 45–54 / 55–64 / 65+)
* **Escolaridade adulto (KPI_004):** "Qual seu nível de escolaridade?" (categorias oficiais NTICS)
* **Acessibilidade (KPI_003 – condicional):** 1. "Este projeto contemplou atendimento especial de acessibilidade?" (Sim/Não) 2. Se Sim: "Qual atendimento especial foi contemplado?"
* **Conhecimento temática (KPI_010):** "Em que grau você entende a temática trabalhada?" (0–10)
* **Conhecimento patrocinador (KPI_011):** "Em que grau você conhece/entende o papel do patrocinador?" (0–10)
* **Conhecimento ODS (KPI_012):** "Em que grau você entende os ODS trabalhados?" (0–10)
* **Satisfação (KPI_013/014):** "Qual seu nível de satisfação com a atividade?" (0–10) + aberta curta
* **Atendimento às expectativas (KPI_015):** "Em que grau o projeto atendeu às expectativas?" (0–10) + abertas
* **Novo olhar para o mundo (KPI_017):** escala 0–10 + "o que mudou na sua vida participar desse projeto"
