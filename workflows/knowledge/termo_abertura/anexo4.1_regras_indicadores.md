# Anexo 4.1 - Regras de Indicadores

## Instrução NTICS – Padrão de Indicadores
Padronizar a seleção, coleta e reporte de indicadores em projetos NTICS por meio de 3 camadas: Gerais, Específicos do Projeto e Por ODS.
Regra central: comparabilidade + assertividade + evidência.

## Camada 1 – Indicadores Gerais (obrigatórios em 100% dos projetos)
Indicadores padrões coletáveis em qualquer projeto NTICS. Regra: sempre incluir todos os indicadores gerais do Master.

Exemplos NTICS validados:
* Beneficiários diretos (KPI_001): contagem em campo
* Beneficiários indiretos (KPI_002): diretos × 3 (padrão IBGE)
* Alcance do projeto (KPI_018): diretos + indiretos
* Perfil do público: escolaridade, idade, gênero
* Execução: carga horária total
* Percepção: conhecimento sobre temática, patrocinador e ODS
* Satisfação: alunos e stakeholders
* Cliente: NPS do patrocinador
* Transformação percebida ("Novo olhar")

## Camada 2 – Indicadores Específicos do Projeto
Escolhidos pelo escopo real do projeto. Regra: só sugerir após entender o escopo.

Informações mínimas para sugerir:
* Tipo de projeto
* Público principal
* Entregas e transformação esperada

Exemplo A – Projeto Educacional:
* Aplicabilidade do conteúdo (escala 0–10, % ≥ 8)
* Mudança de hábito/intenção (aberta + tags)

Exemplo B – Projeto de Geração de Renda:
* Renda familiar / nível socioeconômico (faixas)
* Situação ocupacional (empregado/desempregado/informal/autônomo)
* Intenção de empreender (0–10 + aberta)

## Camada 3 – Indicadores por ODS (ONU: ODS → Meta → Indicador)
Regra de seleção: 1 indicador por ODS correlacionado, vinculando a 1 meta.
Projeto com 4 ODS → 4 indicadores ODS.

Regras de qualidade:
* Se não houver forma prática de medir com evidência, não entra como KPI (pode virar narrativa).
* O indicador ODS deve ser operacionalizado: pergunta / métrica / fonte / evidência.

## Comportamento padrão
1. Sempre iniciar com Camada 1 (Gerais). Calcular: Indiretos = Diretos × 3; Alcance = Diretos + Indiretos.
2. Antes de sugerir Camada 2, validar escopo. Se faltar informação, solicitar.
3. Antes de sugerir Camada 3, validar ODS correlacionados.
4. Para qualquer indicador sugerido, sempre entregar: Nome + definição, Pergunta(s)/método, Métrica/fórmula, Momento, Responsável, Evidência mínima.
