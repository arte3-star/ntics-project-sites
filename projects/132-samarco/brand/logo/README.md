# Logo — Estação Samarco

Logo oficial composto por duas palavras em script (fonte Brotherhood):
- **"Estação"** em azul escuro `#0f406b` — posição superior, sobreposta
- **"Samarco"** em azul claro `#4bc1eb` — posição inferior, em diagonal ascendente

Lockup horizontal sobreposto (as duas palavras se cruzam visualmente — ver página 2 e 3 do manual).

## Variações que devem existir nesta pasta

- `logo-estacao-samarco-cor.png` / `.svg` / `.ai` — versão oficial (azul escuro + azul claro)
- `logo-estacao-samarco-branco.png` / `.svg` — para fundos escuros (todo branco)
- `logo-estacao-samarco-monocromatico-azul.svg` — versão em uma cor `#0f406b`
- `logo-estacao-samarco-preto.png` / `.svg` — versão preta (uso técnico, fax, monocromia)

## Área de segurança (do manual, página 3)

Manter respiro horizontal e vertical equivalente à altura da letra "S" maiúscula de "Samarco". Não invadir essa zona com texto, foto ou outros elementos.

## Regras de uso

- Não distorcer (não esticar, não comprimir)
- Não rotacionar fora do ângulo original
- Não trocar as cores do lockup oficial
- Não aplicar sombra, contorno ou efeito sem aprovação
- **Em peças co-assinadas com Samarco (patrocinador):** o logo institucional Samarco mantém precedência hierárquica conforme régua de marcas
