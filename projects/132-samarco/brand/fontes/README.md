# Tipografia — Estação Samarco

Extraído do Manual de Aplicação Visual oficial.

## Famílias

### Brotherhood (script / handwriting)

- **Uso:** logo "Estação Samarco", títulos display, peças de impacto
- **Onde aparece:** logo oficial (página 2 do manual), capas, headlines de carrossel/post
- **Característica:** caligráfica, traço solto — transmite proximidade e movimento
- **Observação:** o manual aponta Brotherhood como "fonte para textos", mas pelo desenho (script) ela só funciona em **títulos curtos e logo**. Não usar em corpo de texto.
- **Onde baixar:** fonte comercial — pedir ao designer (Marina/Gleidson) o arquivo `.otf`/`.ttf` e colocar nesta pasta

### Museo Sans (sans-serif geométrica)

- **Uso:** títulos de documentos, vídeos, apresentações, corpo de texto, UI
- **Pesos disponíveis no manual:** 5 pesos (Light → Black, conforme amostras do PDF)
- **Pesos recomendados:**
  - Light/Regular → corpo de texto
  - Medium → subtítulos, labels
  - Bold/Black → títulos, ênfase
- **Onde baixar:** fonte comercial (Adobe Fonts) — colocar `.otf`/`.ttf` aqui

## Hierarquia recomendada

| Elemento | Fonte | Peso | Tamanho ref. |
|---|---|---|---|
| Logo / lockup | Brotherhood | — | — |
| Título display (capa) | Brotherhood OU Museo Sans Black | — | 48–96pt |
| Título seção | Museo Sans Bold | 700 | 24–36pt |
| Subtítulo | Museo Sans Medium | 500 | 16–20pt |
| Corpo de texto | Museo Sans Regular | 400 | 12–16pt |
| Legenda / apoio | Museo Sans Light | 300 | 10–12pt |

## Fallbacks web (quando self-host não for viável)

- Substituto de **Brotherhood**: `Caveat`, `Dancing Script`, `Sacramento` (Google Fonts) — ordem de preferência
- Substituto de **Museo Sans**: `Nunito Sans`, `Mulish`, `Source Sans 3` (Google Fonts) — ordem de preferência
