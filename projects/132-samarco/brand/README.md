# Identidade Visual — Projeto 132 (Estação Samarco)

Pacote de identidade visual derivado do KV oficial do projeto. Sempre que for produzir conteúdo (site, PPT, post, carrossel, peça impressa), consultar esta pasta ANTES de gerar qualquer coisa.

## Estrutura

- [kv/](kv/) — KV oficial recebido do designer (PDF/PNG/AI). Fonte da verdade. Não editar.
- [logo/](logo/) — Logos do projeto em variações (cor, branco, preto) e formatos (PNG, SVG, AI).
- [fontes/](fontes/) — Arquivos de fonte (.ttf/.otf) + nome da família e pesos usados.
- [icones/](icones/) — Biblioteca de ícones temáticos do projeto (PNG e SVG).
- [paleta/](paleta/) — Cores oficiais com códigos HEX/RGB/CMYK e regras de uso.

## Precedência de marca

1. **Patrocinador (Samarco)** — logo soberana, paleta do patrocinador prevalece quando o KV define
2. **Identidade do projeto 132** — derivada do KV (esta pasta)
3. **NTICS** — assinatura institucional (ver [brand/NTICS-cheatsheet.md](../../../brand/NTICS-cheatsheet.md) na raiz)

Em caso de conflito entre KV do projeto e padrão NTICS, o KV vence (regras documentadas em `SecondBrain/projetos/132-samarco/brand-aplicacao-samarco.md`).

## Como usar

Quando o Lucas pedir um conteúdo:
1. Ler `paleta/cores.md` para HEX
2. Ler `fontes/README.md` para tipografia
3. Pegar logo correto em `logo/` (qual variação combina com o fundo)
4. Pegar ícones relevantes em `icones/`
5. Conferir o KV em `kv/` para entender intenção e composição
