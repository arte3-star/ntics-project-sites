# Ícones / Elementos Visuais — Estação Samarco

O KV define **bibliotecas de elementos ilustrados por trilha/categoria**. Estilo geral: ilustração vetorial flat, traços orgânicos, com bochechas rosadas nos personagens, fundo arco/colina em cor sólida da paleta atrás de pessoas, paleta fiel às cores do projeto.

## Categorias confirmadas no manual

### Beleza (página 6 do KV)
Personagens femininas com pele variada (afro, branca, ruiva). Itens: batom, esmalte, espelho de mão, pincel de maquiagem, paleta de sombras, prancha de cabelo, secador, máscara facial, soro, tesoura, escova, perfume, pote de creme, estrelas decorativas, necessaire/maleta.
**Fundos arco:** roxo `#6d5ba4`, magenta `#c43f62`, azul claro `#4bc1eb`.

### Culinária (página 7 do KV)
Personagens com chapéu de chef e avental. Itens: panela, tábua de corte, liquidificador, batedeira, espátulas, colher de pau, saleiro/pimenteiro, bule, ervas (alho, cebola, brócolis, cenoura, tomate, abobrinha, beterraba, pimenta), tigela, prato de salada, ralador, luvas de forno.
**Fundos arco:** rosa claro, salmão, magenta `#c43f62`. Verde `#427459` em chapéus/aventais.

## Estrutura sugerida nesta pasta

```
icones/
├── beleza/
│   ├── personagem-1.svg       (mulher afro com creme)
│   ├── personagem-2.svg       (ruiva no espelho)
│   ├── personagem-3.svg       (loira com secador)
│   ├── batom.svg
│   ├── esmalte.svg
│   ├── ... (cada item separado)
├── culinaria/
│   ├── personagem-chef-1.svg
│   ├── ... (cada item separado)
└── (futuras trilhas)
```

Disponibilizar cada elemento em **SVG** (vetor editável) E **PNG transparente** 1000px.

## Regras de uso

- **Manter o estilo:** todos os ícones de uma peça devem ser do mesmo set (não misturar Beleza com Culinária na mesma composição salvo se for um KV multi-trilha)
- **Cores fiéis ao manual:** não recolorir os ícones fora da paleta oficial
- **Fundo arco:** quando usar personagem, manter o arco colorido atrás (é parte da identidade)
- **Composição:** distribuir orbitando o personagem central, com respiro — nunca sobrepor o logo
