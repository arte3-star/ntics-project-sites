Leia e execute o workflow completo em `workflows/marketing/team_social_media.md`.

Use Agent Teams para criar o time. Monte os teammates conforme descrito no workflow:
- **Lead** (social-lead): coordena editorial e estrategia por plataforma
- **Content Writer**: escreve legendas, posts, emails, scripts
- **Publisher**: monta newsletter HTML, cria drafts Gmail, cria tasks ClickUp

Pergunte ao usuario quais plataformas (Instagram, LinkedIn, newsletter, etc.) e se ja existem assets visuais prontos em .tmp/entrega/.
