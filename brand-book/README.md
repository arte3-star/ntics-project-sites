# NTICS Projetos — Brand Book v2.0

> Março 2026 | Evolução v1.0 (2020) → v2.0 (2026)

---

## Sobre este Brand Book

Este é o guia completo de marca da NTICS Projetos, documentando a evolução estratégica de uma empresa com **22 anos de história**, **1.060+ projetos de impacto social** e **11.4 milhões de pessoas impactadas**.

### O que mudou na v2.0

| Aspecto | v1.0 (2020) | v2.0 (2026) |
|---------|-------------|-------------|
| **Público** | B2C (comunidades) | B2B (empresas + leis de incentivo) |
| **Paleta** | 3 cores sóbrias | 9 cores vibrantes + neutrals |
| **Visual** | Filtros opacos | Blocos geométricos coloridos |
| **Posicionamento** | Agência de projetos | Parceira estratégica de impacto ESG |
| **Governança** | Hierárquica | Horizontal, IA integrada |
| **Formato** | PDF estático | Site interativo + documentação |

### O que NÃO mudou
- Logo (lâmpada + engrenagens) — preservado 100%
- DNA da marca: Ambientalista, Legado, Humana
