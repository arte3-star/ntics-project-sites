
### Especificações
- Logo: 120px de largura, versão horizontal
- Fonte: Arial 11px (compatibilidade email)
- Cores: Verde Regeneração para o nome, Grafite para o cargo, Cinza Médio para contato
- Tagline em itálico, Cinza Médio
- Sem blocos geométricos (simplificar para compatibilidade de email)
- Selo Pacto Global: 30px de altura

### Variações
- **Institucional:** inclui selo Pacto Global + ODS
- **Projeto:** inclui logo do projeto + patrocinador (quando aplicável)
- **Simples:** apenas nome, cargo, contato e logo (para comunicação interna)
