## Website

### Diretrizes Gerais
- **Paleta:** predominância de Azul Petróleo + Verde + Branco
- **Tipografia:** Inter (Google Fonts)
- **Blocos geométricos:** presentes em todas as seções como elementos de composição
- **Engrenagens:** decorativas em backgrounds com opacidade reduzida (10-15%)
- **Fotos:** tratamento vibrante, overlay de blocos quando apropriado
- **Navegação:** limpa, com cor de destaque Verde Regeneração

### Páginas Principais

| Página | Propósito | Cor Dominante |
|--------|-----------|---------------|
| Home | Apresentação + impacto + CTA | Azul Petróleo + Verde |
| Projetos | Portfólio com filtros por ODS | Verde + Laranja + Teal |
| Sobre | História + equipe + valores | Azul Petróleo + Branco |
| ESG | Indicadores + relatórios | Teal + Azul Petróleo |
| Contato | Formulário + endereços | Verde + Branco |
| Blog | Conteúdo thought leadership | Branco + acentos variados |

### CTAs do Site
- **Primário:** "Fale com nosso time" → Verde Regeneração
- **Secundário:** "Conheça nossos projetos" → Ghost (outline Azul Petróleo)
- **Campanha:** "Invista com propósito" → Rosa Transformação

## Email Marketing

### Templates

#### Newsletter Mensal
- Header: logo + bloco colorido + título da edição
- Corpo: fundo branco, cards com borda colorida
- CTA: botão Verde Regeneração
- Footer: contato + redes + selos

#### Comunicação de Projeto
- Header: foto do projeto + overlay de bloco com nome
- Corpo: resultados + indicadores + depoimento
- CTA: "Ver relatório completo"

#### Proposta Comercial (email)
- Tom: consultivo e profissional
- Estrutura: contexto → proposta → próximos passos
- Assinatura: completa com selo Pacto Global

### Regras de Email
| Regra | Especificação |
|-------|---------------|
| Largura máxima | 600px |
| Fontes | Arial, Helvetica (webmail safe) |
| Botões | Mínimo 44px de altura, border-radius 8px |
| Imagens | Alt text obrigatório, max 200KB cada |
| Dark mode | Testar e garantir legibilidade |
